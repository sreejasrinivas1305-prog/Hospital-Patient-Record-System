                                                        Hospital Patient Record System (HPRS)

Abstract:
The Hospital Patient Record System (HPRS) is a Python-based Project BasedLearning application developed to manage patient records, ward allocation, billing,
and hospital analytics. The project is divided into 12 weekly modules, beginning with basic patient input andvalidation, and gradually extending to file handling,
object-oriented programming, data analysis using NumPy and Pandas, and visualization using Matplotlib.
The system helps hospitals maintain patient details, search records, assign wards, calculate billing, and analyze patient stay and hospital resource utilization.
The project demonstrates the practical application of Python concepts in a real-world healthcare management system.

Modules Included:
Patient Input
Patient Validation
Patient Search
Patient Functions
Patient Ward Records
Patient OOP
File Handling
Exception Handling
Billing OOP
NumPy Analysis
Pandas Analysis
Visualization

Technologies Used:
Python
NumPy
Pandas
Matplotlib
CSV File Handling
